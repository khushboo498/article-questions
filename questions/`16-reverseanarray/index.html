UNDERSTANDING THE PROBLEM

Well, this problem states that we have to reverse an array. You may refer to the image below for understanding the problem.

Please note that we are not just printing the array in reverse, rather we are reversing it in the memory (as shown in the fig-1) so that the contents of the array are reversed inside it. You may refer to the question video to understand the problem if you face any difficulties.

PSEUDO CODE/APPROACH-1

Create an Auxiliary Array: Create an array of the same size as that of the input array.

Fill the Auxiliary Array: Fill this array by taking the variable at the end of the input array and traversing in reverse direction.

Fill back the original Array: Now copy all the elements back to the original array as it is from the Auxiliary array and the input array has been reversed.

But, friend, we are in the learning phase and it is always better to learn more, right? So, let’s discuss the very basic approach for this problem first. Have a look at the image given below:

So, does this image give you any ideas? Let’s see what we are doing here. We took an array b[ ] which was the same size as that of array a[ ]. Now, we are copying the values of a[ ] into b[ ] in reverse order.

We can do this simply, by taking a variable ‘i’ on the last index of a[ ] and a variable ‘j’ on the index 0 of array b[ ]. We do b [ j ]= a[ i ] and the value of a[ i ] gets copied to b[ j ] .

Then we increment the value of ‘j’ and decrement the value of ‘i’ and repeat the same procedure till ‘j’ completes the array in the forward direction and ‘i' completes the array in the backward direction.

As a result, we will get the reverse of a[ ] in b[ ] . But the problem states that we have to reverse the array a[ ]. This means that we can’t have the reverse of a[ ] just kept stored in b[ ]. Rather we have to reverse a[ ]. So we will copy back the values of b[ ] to a[ ] as shown in the figure. (fig-3).

So, as you can see from the figure, we have reversed the array a[ ] by copying back the elements from b[ ] to a[ ]. Dear reader, Please think that:

Analysis

So, we have traversed the array twice. Once while copying the values from a[ ] to b[ ] and other times while copying back the values from b[ ] to a[ ].

Time Complexity

O(n) as we have traversed the array twice. Once while copying the values from a[ ] to b[ ] and other times while copying back the values from b[ ] to a[ ].

Space Complexity

O(n) as we have used an extra array which is also known as an auxiliary array in the programmer’s language. The size of the array was the same as that of the given array.

PSEUDO CODE/APPROACH-2

  1. Take two variables: Take a variable ‘i’ at the 0th index and j at the end of the array.
  2. Swap:Swap the values arr[i] and arr[j] and increment i and decrement j. Keep up this procedure till i<j and the array will be reversed.

Now that we have seen a basic method to solve the problem, we have also encountered that the space complexity was O(n) due to using the auxiliary array. So, we will avoid using that array. Dear reader, try and think how we can avoid using an extra array and save the memory. Let’s discuss this method now. Have a look at the figure given below:

So, did you understand what we are doing here? Don’t worry if you didn’t. Let’s understand this together. We have kept a variable ‘i' at index 0 of the array and a variable ‘j’ at the last index of the array. Now we are swapping the values a[ i ] and a[ j ] i.e. we are swapping the first and the last value. Then we move our ‘i’ variable one place ahead and bring our ‘j’ variable one place back and swap the values a[ i ] and a[ j ]] again i.e. we are now swapping the second and the second last values. We will continue this process till (i<j) as when ‘i’ becomes greater than or equal to ‘j’ we would have scanned the entire array and reversed it. This is also called reversing in-place. If you have any problem in reversing the array using this method you may refer to the solution video (0:16-0:43). Now let’s have a look at the code for this method.

import java.io.*; import java.util.*; public class Main{   public static void display(int[] a){     StringBuilder sb = new StringBuilder();     for(int val: a){       sb.append(val + " ");     }     System.out.println(sb);   }   public static void reverse(int[] a){           for(int i=0,j=a.length-1,i<j;i++,j--)       {           int temp=a[i];           a[i]=a[j];           a[j]=temp;       }   } public static void main(String[] args) throws Exception {     BufferedReader br = new BufferedReader(new InputStreamReader(System.in));     int n = Integer.parseInt(br.readLine());     int[] a = new int[n];     for(int i = 0; i < n; i++){        a[i] = Integer.parseInt(br.readLine());     }     reverse(a);     display(a);  } }
