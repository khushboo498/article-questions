PROBLEM DISCUSSION:

First of all you will get a number n, which indicates the length of an array. Then you will get n more inputs corresponding to each index of the array.

And you are required to find the span of the array, which is defined as the difference between max and min value of the array.

You may also watch the question video for better understanding of the question.

Before you move further, it is advised that you spend enough time to understand the question.

Approach:
  • In order to solve this problem, we will use a for loop on the input array and find the minimum and maximum values of the array.
  • After finding minimum and maximum values of the array, we will print their difference (maximum value - minimum value) which is the span of the array.
Pseudo Code:
  • First of all you will be writing an input statement for an integer n.
  • This n will now be helpful in defining an array ‘arr’ of length n.
  • Now, you will run a for loop on integer i, initialized to 0, with an increment of one in each iteration and satisfying the condition i < n.
  • And inside this for loop you will be taking input of ith element of array arr.
    Moving further, now in order to find the span of an array, you need to find min and max values of the array. So give it a thought and try to figure out how you can find min and max values.
    It’s okay, if you couldn’t find it, don’t worry; we will look at it together now.
  • We will be needing two variables, min and max, both initialized to arr[0] that is the first value of array arr.
  • A for loop on array arr is applied so that values of min and max can be set to corresponding minimum and maximum values of array using if conditions.
  • The value of min and max gets compared with the arr[i] in each iteration and gets updated simultaneously whenever the condition is satisfied.
  • Min gets updated whenever arr[i] is smaller than min (arr[i]< min) and max gets updated when arr[i] is greater than max (arr[i]>max).
    For better understanding of this part, it is advised to you to watch the solution video from [1:46 to 3:21] .
  • Since we have successfully computed min and max values, it is easy to print their difference.
For example:
  • Let's assume you get n as 6 in input and you initialize an array arr of length 6.
  • Now with the help of a for loop you can allocate the values to each index of the array. Let’s say you assigned 10, 5, 4, 19, 3 and 8 as values to corresponding indices of arr.
  • Initially both min and max variables are initialized to 10 here.
  • Now the next for loop will run which will help us to find the minimum and maximum value of the array.
  • On applying for loop on array, we can see above that the value of min and max gets compared with the arr[i] in each iteration and gets updated simultaneously whenever condition is satisfied.
  • Min gets updated whenever arr[i] is smaller than min i.e. at index 1, 2 and 5 and max gets updated when arr[i] is greater than max i.e. at index 3.
  • At last we have min as 3 and max as 19. So the span in this case will be (max - min = 19 - 3) i.e. 16.
import java.io.*; import java.util.*; public class Main { public static void main(String[] args) throws Exception { Scanner scn = new Scanner(System.in); int n = scn.nextInt(); int[] arr = new int[n]; for (int i = 0; i < n; i++) { arr[i] = scn.nextInt(); } int min = arr[0]; int max = arr[0]; for (int i = 1; i < arr.length; i++) { if (arr[i] < min) { min = arr[i]; } if (arr[i] > max) { max = arr[i]; } } int span = max - min; System.out.println(span); } }

Time and Space Complexity Analysis:
Time Complexity:

O(n)

A ‘for’ loop is used to find the minimum and maximum values in the array; travelling the n sized array makes the time complexity O(n).

SPACE COMPLEXITY:

O(1)

Since no extra space is used, therefore space complexity is constant.
