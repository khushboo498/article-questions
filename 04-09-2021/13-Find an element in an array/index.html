PROBLEM DISCUSSION:
  • Just like the previous question, first of all you will get a number n, which indicates the length of an array. Then you will get n more inputs corresponding to each index of the array.
  • After that you will be given one more input, d as data, for which you have to find, at which index of array d is present. And in case d is not present in the array then you need to print -1.

You may also watch the question video for better understanding of the question.

Before you move further, it is advised that you spend enough time to understand the question and give it a fair try .

Approach:
  • In order to solve this problem, we will use a for loop on the input array and check if a value equivalent to data is present or not.
  • And if an equal value is present then we will print the index of this value and return.
  • And if an equal value is not present then the control will come out of the for loop and we will return -1.
Pseudo Code:
  • First of all we need to write an input statement for an integer n.
  • This n, which represents length of the array will be helpful in defining an array say ‘arr’.
  • Now, you need to run a for loop on the array with integer i, initialized to 0. Also satisfying the condition i < n and with an increment of one to i, in each iteration and inside this for loop you will be taking input for ith element of array arr.
  • After that take input for d (data); the value for which we need to find the index if the similar value is present in the array.
    Our aim is to find the index of data in array arr, so give it a thought and try to figure out how you can find it.
    It’s okay, if you couldn’t find it, don’t feel bad. Let’s look at it together.
  • So for finding the index of data, it is required that data is present in the array. To check this, you need to apply a loop on the array length.
  • And then using conditional ( if(d == arr[i] ) ), you can make sure that whether data is present in the array or not. Whenever, ( if(d == arr[i] ) ) this condition is satisfied, index i will be printed and the return statement will restrict the further iterations of the loop and stop the execution.
  • And in case, a value equivalent to data is present more than once in a given array, then the first index with value equal to data will be printed.
  • Also in case, where value equivalent to data is not present in the array then, after control comes out of the for loop, -1 will be printed.

So reader, if you still face any problem in understanding this part, we advise you to watch [1:47-3:40] portion of the video.

For example:
  • Let's assume, the user enters n as 5 in input and an array arr is initialized of length 5.
  • So with the help of a for loop, values to each index of the array will be allocated. Let’s say you assigned 1, 2, 8, 5 and 3 as values to corresponding indices of arr.
  • After that, a for loop will run on the array until a value equal to data is found.
  • In this case, 8 is present at index 2. So when the for loop will run for i = 2, at that time data will be equal to arr[2], therefore 2 (i) will be printed and then the return statement will be executed.
import java.io.*; import java.util.*; public class Main{ public static void main(String[] args) throws Exception { Scanner scn = new Scanner(System.in); int n = scn.nextInt(); int[] arr = new int[n]; for (int i = 0; i < n; i++) { arr[i] = scn.nextInt(); } int d = scn.nextInt(); for(int i = 0; i < arr.length; i++){ if(d == arr[i]){ System.out.println(i); return; } } System.out.println(-1); } }

Time and Space Complexity Analysis:
Time Complexity:

O(n)

A ‘for’ loop is used to find an equivalent value in the array; travelling the n sized array makes the time complexity O(n).

SPACE COMPLEXITY:

O(1)

Since no extra space is used, therefore space complexity is constant.
