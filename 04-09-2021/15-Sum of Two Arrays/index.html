For explanation we take the input array as [1,2,3,4,5] . We consider each element as the height of a building and find what the largest height is for making a rough figure. Here, it is 5.

We take a grid that looks something like this where the vertical line depicts the floor of the building and horizontal line depicts the height of each building:

Now we decide if on the 5th floor of 1st building, would there be a brick (Star) or not. So in this case, there would not be a brick. So we represent it as a dash. Pseudo Code:
  • Find out “max”, the maximum of all the elements.
  • The outer loop runs for each “floor” from max to 1.
  • The inner loop runs for each element of the array or “height” from index 0 to arr.length-1.
  • If arr[i]/ height> floor then print star, else print blank.
import java.io.*; import java.util.*; public class Main{ public static void main(String[] args) throws Exception { Scanner scn= new Scanner(System.in); int n=scn.nextInt(); int[]arr=new int[n]; for(int i=0;i< arr.length;i++){ arr[i]=scn.nextInt(); } int max=arr[0]; for(int i=1;i< arr.length;i++){ if(arr[i]>max){ max=arr[i]; } } for(int floor=max;floor>=1;floor--){ for(int i=0;i< arr.length;i++){ if(arr[i]>=floor){ System.out.print("*\t"); } else{ System.out.print("\t"); } } System.out.println(); } } }
Java; true";
Analysis

We are sure you must have been impatient for the “HOW” and “WHY” of this code. Slow down kiddo, patience is the essence of coding.

Let’s understand this code in 2 parts.

  1. In this code, we find out the maximum of all the elements by assigning max with the value at 0 index and then comparing it with the subsequent values.
  2. Here my dear coder, we observe that the variable ‘floor’ denotes the floor of the building and variable ‘i’ denotes the number of the building. Hence, arr[i] means the height of the building no. ‘i’ .
  1. The outer loop runs for each floor from max of the heights to the 1st floor.
  2. The inner loop runs for each element of the array from index 0 to arr.length-1
  3. For a given iteration if the height of the building (Or the total number of floors the building has) is more than the ‘floor’ value then we print a start there else we print a blank(tab).
Time and Space Complexity Analysis:
Time Complexity:

O(n2)

This time complexity is quadratic because nested for loops are used.

SPACE COMPLEXITY:

O(n)

As a 1D array is used, therefore space complexity is linear.
